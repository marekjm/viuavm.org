Code and resources

Where to find code and other resources for Viua VM.

Code

Code is hosted both at Savannah (more information here), and at Github. Both repositories are kept up-to-date so you can clone from either.

CI

Continuous integration is running at ci.viuavm.org.

Releases and downloads

Release tarballs can be downloaded from Savannah (fetch .tar.gz with newest tag) and Github.

Bugtracker

Issue tracker at Github is used for bug and feature tracking in Viua VM.

Mailing lists

Mailing lists are hosted at Savannah.

News and announcements

On Diaspora* the #viuavm tag can be followed to receive news and announcements from Viua VM. There is also the news section at Savannah (and a related Atom feed). Additionally, you can follow the Viua VM Weekly blog to read weekly updates covering developments in Viua VM.
