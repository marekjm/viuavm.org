Code and resources

Where to find code and other resources for Viua VM.

Code

Code is hosted both at Savannah (more information here), and at Github.

CI

Continuous integration is running at TravisCI.

Releases and downloads

Release tarballs can be downloaded from Savannah (fetch .tar.gz with newest tag) and Github.

Bugtracker

There is no centralised bugtracker, yet. Either the one at Savannah may be used, or you can open an issue at Github.

Mailing lists

Mailing lists are hosted at Savannah.

News and announcemnts

News and announcements are made on Diaspora* (follow #viuavm on your pod), and at Savannah (there is also an Atom feed you can subscribe to). Release notes are also automatically generated by Github (it displays tag message as release notes).
